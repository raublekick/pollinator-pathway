# Pollinator Guide

This project adapts the Maricopa County Seed Library's Maricopa Pollinator Pathway program into a searchable web application. Users can select plants and verify adherance to the various certification levels.

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Type-Check, Compile and Minify for Production

```sh
npm run build
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```
